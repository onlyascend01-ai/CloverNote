Installation Guide

CloverNote is a native Windows application. Follow these steps to get set up.

1

Download the .exe installer from the home page.

2

Run the installer. Windows might show a "SmartScreen" warning—click "More Info" and then "Run Anyway".

3

The app will launch automatically once installed.

How the Vault Works

Unlike cloud-based notepads, CloverNote stores everything in a "Local Vault" on your hard drive.

Data Location

Your notes are stored in your AppData folder. They are encrypted if you have set an account password.

No Cloud Sync

We do not sync to servers. This means your data is 100% private, but you are responsible for backups.
